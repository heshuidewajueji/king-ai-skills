# Claude App Builder

This skill provides a complete workflow for building production-ready applications using Claude Code.

## Workflow Phases

### 1. Idea (`/idea`)
Define the application concept, target users, and core features.

### 2. Plan (`/plan`)
Create detailed implementation plan with file structure and dependencies.

### 3. Feature (`/feature`)
Build individual features with proper testing and documentation.

### 4. Implement (`/implement`)
Execute implementation following best practices.

### 5. Test (`/unit-test`, `/e2e-test`)
Write comprehensive tests (unit and end-to-end).

### 6. Review (`/review`)
Code review with quality and security checks.

### 7. Status (`/status`)
Track project progress and identify blockers.

## Usage

```bash
"Build a full-stack app with claude-app-builder workflow"
"/idea Create a note-taking app with offline support"
"/plan Design the API architecture"
```

- Follow frontend.md for UI best practices
- Follow testing.md for test coverage requirements
